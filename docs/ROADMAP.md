# Roadmap

## Phase 1 - Foundation
- stabilize project structure
- define outputs and schemas
- implement collector orchestrator
- implement common helpers
- implement quick mode

## Phase 2 - Deep collection
- add service/process correlation
- add DNS/TLS mapping
- add users/access/persistence modules
- add containers and database modules
- add intelligent file inventory

## Phase 3 - Analysis
- parse collector outputs
- normalize objects
- build relationships
- compute risk and maintainability scores
- emit markdown and JSON reports

## Phase 4 - Diff and baseline
- store baseline snapshots
- compare current run to previous run
- report drift, additions, deletions and critical changes

## Phase 5 - Remediation preparation
- generate prioritized remediation candidates
- add precaution notes before touching critical components
- prepare reconstruction-oriented summaries
